## Usage
- Make sure to have at least Zig `0.12.0-dev.2668+ddcea2cad`
- Open a terminal and run `build.bat run` on `Windows` and `build.sh run` on `linux`.

> [!NOTE]
> Run `build.(bat/sh) -h` for more options.
